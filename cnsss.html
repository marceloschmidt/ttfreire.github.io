About the game.

Copa Toon (or Cartoon Network Superstar Soccer in the US) is a fast-paced online soccer game starring characters from Adventure Time, Regular Show, The Amazing World of Gumball and Ben 10. The game features single and multiplayer modes.

My work in this project.

Cartoon Network Superstar Soccer is a sychronous multiplayer online game, meant to be played on mobile devices. This was a big challenge, since our team was small and we had to make sure the game would run smoothly on several devices. The game was a success and we had the opportunity to create a sequel in 2016.
